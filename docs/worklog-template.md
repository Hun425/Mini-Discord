# 프로젝트 개요
이 문서는 미니 디스코드 프로젝트 작업 시 기록을 남길 수 있는 워크로그 템플릿입니다. 각 단계를 기록하여 나중에 과정을 쉽게 재현할 수 있도록 합니다.

# 사용 방법
1. 아래 단계별 템플릿을 복사해 사용합니다.
2. 실행한 명령어와 Codex 프롬프트, 생성된 코드나 매니페스트, 실행 로그를 기록합니다.
3. 오류가 발생했다면 실패 섹션에 상세히 적고, 회고 내용을 남깁니다.
